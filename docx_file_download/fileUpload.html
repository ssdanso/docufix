NOTE:

Docufix only supports file formats like .csv, .json, .xml .txt .pdf and .docx for uploading

If you must compare documents from any other format, aside the above formats mentioned,
please make use of the text option, then copy and paste into the text-box.

Your compared data gives you result in percentage %

You can choose to download compared documents as .doc or pdf file formats

This App has an inbuilt words formatting editor

Compared texts are editable, this means you can edit your work for correction after comparing.

IMPORTANT!!!
    The following activities would not be efficiently carried out on mobile phone except for larger screen sizes:
  • Comparing od documents
  • Formatting of compared document
    Also note the meaning of the highlighted outputs:
  • RED HIGHLIGHT Means the texts in second text input that are not found in the first text input
  • YELLOW HIGHLIGHT Means the texts in first text input that are not found in the second text input
  • PLAIN TEXT Means text found in both tex inputs
